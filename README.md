# largefish

> just a demo framework extend [umi](https://github.com/umijs/umi)

## Start

```bash
yarn
yarn build -w
```

link largefish

```bash
cd packages/largefish && yarn link
```

use in project

```bash
# dev
largefish dev

# build
largefish build
```

## Test

```bash
yarn test
```

## Publish

```bash
yarn publish
```
